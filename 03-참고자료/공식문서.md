## 읽을 때 주의할 점

- React 공식 문서는 "state를 최소로 둔다", "Effect는 외부 시스템과 동기화할 때 쓴다"는 관점이 반복됩니다.
- 성능 문서는 `memo`, `useMemo`, `useCallback`을 먼저 쓰라는 뜻이 아니라, 측정 후 필요한 곳에 쓰라는 뜻으로 읽습니다.
- React Compiler 문서는 수동 memoization을 줄이는 흐름을 설명하지만, 상태 설계와 컴포넌트 책임 분리는 여전히 개발자가 판단해야 합니다.
