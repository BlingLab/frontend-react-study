# frontend-react-study

React를 읽고, 고치고, 다시 설명하기 위한 프론트엔드 학습 저장소입니다.

출퇴근 시간에는 `01-학습문서/`를 책처럼 읽고, 책상 앞에서는 Vite 실습 앱을 실행해 작은 예제를 직접 바꿔봅니다. 단순히 공부한 내용을 쌓는 공간이 아니라, React를 단계별로 이해하고 다시 설명하기 위한 정리형 학습 저장소입니다.

## 이 저장소를 쓰는 방법

```bash
pnpm install
pnpm run 예제:기초
```

1. [문서 목차](./01-학습문서/README.md)에서 오늘 볼 난이도를 정합니다.
2. 이동 중에는 `01-학습문서/01-기초/`부터 순서대로 읽습니다.
3. 책상 앞에서는 같은 단계의 `02-학습예제/` 문서를 보고 예제를 실행합니다.
4. 헷갈린 개념은 각 단계 README의 "빠른 정리"에서 다시 확인합니다.
5. 더 정확한 근거가 필요하면 [03-참고자료](./03-참고자료/README.md)에서 공식 문서를 확인합니다.

## 학습 원칙

- API 이름보다 "왜 이 상태가 여기 있어야 하는가"를 먼저 설명합니다.
- `useEffect`는 렌더링 계산용이 아니라 외부 시스템과 동기화할 때 사용합니다.
- `useMemo`, `useCallback`, `memo`는 측정 후 필요한 곳에만 적용합니다.
- 예제는 작게 유지하고, 문서는 난이도 순서대로 자연스럽게 이어지게 관리합니다.
- 읽은 내용은 반드시 내 말로 다시 적습니다.
